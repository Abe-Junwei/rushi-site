- 公开版本对齐 **v1.0.0**，安装包与 OTA 清单托管于 [updates.rushi.app](https://updates.rushi.app/latest.json)（不再依赖 GitHub Release 挂载安装包）。
- **Windows**：推荐下载离线完整包 zip（约 1.4GB），完整解压后运行同级安装包；附带 bundled ASR 模型资源；可选单独下载安装包 exe 作 OTA / 补装。
- **macOS**：提供 Apple Silicon unsigned DMG；首次打开需按 Gatekeeper 流程（Control-点击打开，必要时清除 quarantine）。
- 本机 FunASR 侧车首启 seed 后可断网转写；在线 STT 仍为可选 Provider。
- Editor 支持波形对齐校对、拆分 / 合并 / 冻结；定稿模式含规则纠错、智能改稿与大模型润色（Word 修订轨）。
- 术语表与纠错记忆可提高专名稳定性；导出 TXT / SRT / DOCX，场次元信息可进入文档。
- 应用内更新（Tauri 验签）已接入 macOS 与 Windows；验签失败请改从 CDN 手动安装。
- Windows NVIDIA 用户可在环境页按需下载 CUDA 侧车（非首装必需；侧车未就绪时仍可用 CPU）。
- 变速播放尽量保持音高，便于听写校对。
