- 公开版本对齐 **v1.0.0**，安装包与应用内更新由官方 CDN 分发，详见[下载页](/download)。
- **Windows：**推荐下载离线完整包 zip（约 1.4 GB），完整解压后运行同级安装包；内含本机转写模型资源。也可单独下载安装包 exe，用于补装或应用内更新。
- **macOS：**提供 Apple Silicon DMG；首次打开若系统提示未识别开发者，请在 Finder 中按住 Control 点击应用再选择“打开”。
- 本机转写在首次准备模型后可断网使用；在线转写为可选能力，需自行配置厂商服务。
- 编辑器支持波形对齐校对、拆分、合并、冻结；定稿模式含规则纠错、智能改稿与大模型润色（Word 修订轨）。
- 术语表与纠错记忆可提高专名稳定性；导出 TXT、SRT、DOCX，场次元信息可进入文档。
- 应用内更新已接入 macOS 与 Windows；若更新失败，请改从下载页手动安装。
- Windows NVIDIA 用户可在环境页按需下载 CUDA 侧车（非首装必需；未就绪时仍可用 CPU）。
- 变速播放尽量保持音高，便于听写校对。
