# 本机 ASR

Rushi 桌面端会拉起本机 ASR sidecar（默认 `http://127.0.0.1:8741`）。安装包随附或首启 seed 默认模型后，可在断网状态下进行本机转写。

## 首次准备

1. 安装完成后启动应用，按向导完成模型 seed。
2. 打开 **设置 → 本机 ASR / 环境**，确认侧车与模型状态可用。
3. 若显示休眠或未就绪，按提示「应用并重启侧车」。

## Windows GPU（可选）

有 NVIDIA 显卡时，可在环境页下载 CUDA 侧车以加速本机转写。CUDA 为 CDN 可选组件，**不在**离线安装包主路径内；下载失败或未安装时仍可继续 CPU 转写。

## 排障

转写失败时查看编辑器内转写诊断，或环境页本机 ASR 状态。仍无法恢复时，导出诊断包（zip）便于排查。更多安装与报错说明见[使用说明 · 安装与首次启动](/docs/getting-started)。
